**Multiple query execution** combines multiple GraphQL queries together and executes them as a single operation.

It improves the application's performance, as it avoids the latency from waiting for the response from executing multiple requests, and organizes the code into logical units.

_In the video: A field returns a different response depending on the logged-in user being an admin or not._
