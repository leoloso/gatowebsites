**Retrieve data from, and post data to, any external service** that offers an API.

For instance, you can interact with Notion, AirTable, GitHub, Mailchimp, Zapier, OpenAI, ConvertKit, and many more.

Any kind of API is supported, and convenient methods for REST and GraphQL APIs are provided.
