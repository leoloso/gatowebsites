Gato GraphQL provides a **GraphQL server** for your WordPress site. Use **GraphQL** to retrieve exactly the data needed to power your WordPress themes, plugins, blocks and applications.

Provide a GraphQL query, and execute it against the server to retrieve the data, in JSON format.
