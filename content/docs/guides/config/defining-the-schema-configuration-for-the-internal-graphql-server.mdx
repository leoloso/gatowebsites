# isPRO: true

---

The **[Internal GraphQL Server](/extensions-reference/internal-graphql-server)** by default applies the Schema Configuration defined in its own Settings.

The schema configuration for the **Internal GraphQL Server** is selected in the Settings page, under tab "Server Configuration > Internal GraphQL Server > Schema Configuration":

<Image
  src="/assets/guides/upstream-pro/settings-schema-configuration-for-internal-graphql-server.png"
  width="1024"
  height="940"
  alt="Schema configuration for the Internal GraphQL Server in Settings"
  caption="Schema configuration for the Internal GraphQL Server in Settings"
/>

To make changes, select a new schema configuration, and click on `Save Changes`.
