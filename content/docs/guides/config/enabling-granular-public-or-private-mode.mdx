In the settings for module "Public/Private Schema", there is option "Enable granular control?":

<Image
  src="/assets/guides/upstream-pro/settings-enable-granular-control.png"
  width="1024"
  height="940"
  alt="Enable granular control?"
  caption="Enable granular control?"
/>

When this value is `on`, when editing an entry in an [access control list](/guides/use/defining-access-control), we can select individually on fields and directives if they are public or private:

<Image
  src="/assets/guides/upstream-pro/public-private-individual-control.png"
  width="400"
  height="220"
  alt="Individual Public/Private schema mode"
  caption="Individual Public/Private schema mode"
/>
