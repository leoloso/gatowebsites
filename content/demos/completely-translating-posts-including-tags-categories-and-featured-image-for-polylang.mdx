New translations for all defined languages will be automatically created for:

- The post's content
- Categories
- Tags
- Featured image

It works using the **Polylang** plugin (or **Polylang PRO**) and connecting to the **Google Translate API**, and it supports both the Gutenberg block editor and the Classic editor.
