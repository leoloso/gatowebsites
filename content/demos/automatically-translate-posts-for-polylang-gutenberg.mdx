We can use Gato GraphQL to automatically translate posts por the **Polylang** plugin (and **Polylang PRO**), using the **Google Translate API**, with support for the block editor.

Please watch the demo video, in which Polylang is configured with 3 languages: English (default), Spanish and French.

We first translate a post to all languages by running [persisted query **Translate posts for Polylang (Gutenberg)**](library/translate-posts-for-polylang-gutenberg) (already provided by the Gato GraphQL plugin).

Then we automate the process, by publishing automation rule **Polylang: When publishing a post, translate it to all languages (Gutenberg)** (already available by Gato GraphQL PRO, but as `draft`), which translates the post automatically (whenever the `publish_post` hook is triggered).
