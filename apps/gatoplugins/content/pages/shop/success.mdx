This purchase, and all previous orders, can be accessed via [My Orders](/shop/my-orders).

## Installing the plugin in WordPress

Download the plugin by Gato Plugins from the link in the email.

Install it in your WordPress site (under `Plugins > Add new > Upload`), and activate it.

Upon activation, an admin notice will request to provide the license key:

<Image
  src="/assets/shop/install-product-admin-notice.png"
  width="1024"
  height="940"
  alt="Admin notice requesting to provide the license key"
  caption="Admin notice requesting to provide the license key"
/>

Click on the link to go to the Settings page. Then copy/paste the license key, and click on `Activate Licenses (or Deactivate/Validate)`.

<Image
  src="/assets/shop/install-product-enter-license-key.png"
  width="1024"
  height="940"
  alt="Entering the license key"
  caption="Entering the license key"
/>

The plugin will now be activated and enabled:

<Image
  src="/assets/shop/install-product-activated.png"
  width="1024"
  height="940"
  alt="Plugin activated"
  caption="Plugin activated"
/>

## Upgrading the plugin (when a new version is released)

While your purchased plugin's license is active, you can access its latest version in LemonSqueezy's <a href="https://app.lemonsqueezy.com/my-orders" target="_blank">My orders</a> page.

Whenever a new version of the plugin is released, download it and install it on your WordPress site.
